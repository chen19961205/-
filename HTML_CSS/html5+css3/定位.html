        偏移后不会影响兄弟盒子位置，不脱离标准流，文档继续保留他原来的位置

绝对定位

1.如果没有祖先元素或者祖先元素没有定位，则以浏览器为准定位
         2.如果祖先元素有定位(相对，绝对，固定),则与最近一级的有定位祖先元素为参考点移动位置
         3.会脱标准流，下面的兄弟盒子B会占据原来A盒子的位置，不占空间

粘性定位sticky

        1.以浏览器的可视窗口为参照点移动元素
        2.占有原来位置
        3.必须添加top left right bottom中的一个才起作用
粘性定位

固定定位fixed

1.在可视区域内变化，以浏览器可视窗口为参照点
        2.无论祖先元素有没有定位，都会以浏览器为准，不随着滚动条滚动而滚动，适合浏览器中的广告设置
        3.脱离标准流，不占用位置，会覆盖其他元素
        4.固定在版心右侧位置（重要），返回到首页
            4.1让固定盒子left50%，然后设置margin
